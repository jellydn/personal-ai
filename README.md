# personal-ai

> Public original projects created during the last nine months, built with AI-assisted development.

This collection covers public original repositories created between **October 20, 2025 and July 20, 2026**. Forks and private repositories are excluded.

Over the last nine months, I used AI-assisted development to build around 30 public projects, including complete web applications, AI agents, RAG systems, developer tools, editor extensions, local-model utilities and productivity apps.
